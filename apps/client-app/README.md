# client-app

`client-app` is the future Flutter application for Lenker end users.

Supported `MVP v0.1` targets:

- Android
- Windows
- macOS

Planned `MVP v0.1` scope:

- email-first sign-in
- provider deeplink or provider code onboarding
- subscription sync
- connect and disconnect flow
- region selection
- basic diagnostics
- key rotation
- encrypted local storage

This directory currently contains only minimal Flutter project metadata and does not include application logic yet.
