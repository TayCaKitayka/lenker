# Code of Conduct

Lenker is an open-source project. Participation should stay respectful,
practical, and focused on building a trustworthy system.

## Expected Behavior

- Be respectful of different experience levels, backgrounds, and viewpoints.
- Keep technical disagreement specific and evidence-based.
- Assume public project spaces are shared working areas.
- Report security issues responsibly instead of publishing exploit details.
- Avoid harassment, threats, personal attacks, or discriminatory language.

## Unacceptable Behavior

- Harassment, intimidation, or abuse.
- Sexualized language or unwanted attention.
- Publishing private information without consent.
- Deliberate disruption of issues, pull requests, or project discussions.
- Using project spaces to promote fraud, malware, or deceptive VPN services.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block
participants when behavior harms the project or its community.

This code of conduct is inspired by the Contributor Covenant, version 2.1, but
kept intentionally short for the current project stage.
